By default we only push asset that are ONLY defined modes. It detect by your modes name and push. The rest of the modes will not be available in your browser. This is due to performance where Ace Editor contains more than 10 js files for all modes. If you really want to do by scripting that can switch themes or maybe other modes via scripting , you have to push ALL ACE's JS files in order to do that. Here is the code : ```javascript // In modules/custom-code-editor-a3/index.js module.exports = { options: { ace : { // all other ace options... scripts : { pushAllAce : true } } } } ``` > NOTE: Beware that this push ALL ACE JS files including your own mode.
